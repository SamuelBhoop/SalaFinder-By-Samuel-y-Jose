# SalaFinder

Aplicación web para la reserva de salas y espacios de trabajo.

## Tecnologías

- React 19 + Vite
- React Router v7
- Tailwind CSS v4

## Instalación

```bash
npm install
npm run dev
```

La app estará disponible en `http://localhost:5173`

## Páginas

- `/login` — Inicio de sesión
- `/spaces` — Listado de espacios disponibles
- `/spaces/:id/reserve` — Formulario de reserva
- `/my-reservations` — Mis reservas
- `/calendar` — Calendario de reservas
